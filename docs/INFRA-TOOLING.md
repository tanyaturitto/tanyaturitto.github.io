## Quick Reference — Tech Stack

| Layer | Technology | Purpose |
|-------|-----------|---------|
| Structure | HTML5 | Semantic page markup |
| Styling | CSS3 Vanilla | Layout, theming, dark mode |
| Interactivity | JavaScript ES6+ | Dark toggle, scroll reveal |
| Typography | Google Fonts | Lora + Source Sans 3 |
| Domain | Porkbun — wisebuilder.me | Domain registration & DNS |
| Email | Cloudflare Email Routing | Tanya@wisebuilder.me → Gmail |
| Hosting | GitHub Pages | Free static hosting, auto-deploy |
| Version Control | Git + GitHub | Source control & project board |
| Editor | VS Code | Local development |
| AI Assist | Claude — Anthropic | Build, audit, documentation |
