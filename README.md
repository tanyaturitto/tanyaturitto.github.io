## 🛠️ Project #1 — Resume Portfolio Site

A WCAG 2.1 AA compliant single-page portfolio site built with vanilla 
HTML, CSS, and JavaScript.

**Key features:**
- WCAG 2.1 AA accessibility compliant
- Light / dark mode toggle
- Fully responsive
- Zero dependencies — no frameworks, no build step
- Hosted free on GitHub Pages with custom domain

**Tech stack:** HTML5 · CSS3 · JavaScript ES6+ · GitHub Pages · Porkbun · 
Cloudflare Email Routing
